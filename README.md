# 🌱 AI Crop Recommendation Agent (Agentic AI + Machine Learning)

An intelligent **Agentic AI system** that recommends suitable crops based on **real-time weather data**, **season detection**, and **machine learning predictions**.

---

## 🚀 Project Overview

This project combines:

* Agentic AI (LangGraph)
* Real-time Weather API
* Machine Learning (Random Forest)
* Smart Agriculture Decision Support
* Streamlit Interactive UI

The system helps farmers and researchers choose crops according to climate conditions in Pakistan.

---

## 🧠 How It Works

1. User enters a city name
2. AI agent fetches live weather data
3. Season is detected (Rabi / Kharif / Zaid)
4. ML model predicts best crops
5. Recommendations displayed via Streamlit UI

---

## ⚙️ Technologies Used

* Python
* LangGraph
* Streamlit
* Scikit-learn
* Pandas & NumPy
* WeatherAPI
* Random Forest Classifier

---

## 📊 Machine Learning Model

**Algorithm:** Random Forest Classifier

**Input Features:**

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

Model accuracy achieved: **~99% (dataset dependent)**
